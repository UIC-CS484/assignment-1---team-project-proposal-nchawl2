FootWiz: Comprehensive Soccer Dashboard

RESTful API: API-FOOTBALL

About:
The API contains information about 880+ football leagues and cups worldwide. Users can obtain information about the pre-match odds, match events, lineups, coaches, players, standings, statistics, and transfers.

Data Collection:
The site will collect user information, such as user name and email to create a user profile. Other information that may be collected would include location, favorite leagues, favorite teams, etc.

Database Entities:
  • USER -> firstName, lastName, email, password
  • LEAGUE -> leagueName, country, teamName
  • PLAYER -> firstName, lastName, leagueName, teamName, nationality, position, age

About The Project

Potential Users:
The website will be primarily used by people looking to follow the latest information about the top soccer leagues across the world. Users will be able to follow teams and leagues to get information such as standings, match dates, lineups, etc.

Site Goals:
  • Create an interactive dashboard with personalised information
  • Present data in a clear manner

Motivation:
As someone who loves the sport, I wanted to create a project that focuses on creating an interactive dashboard with all the information about the leagues and teams I would like as a user. My main aim is to provide information in a concise manner, but also allow the user to dive in and get more detailed information if they would like.
